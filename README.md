# Rastreamento dos correios

O App tem finalidade de fazer o rastreamento de encomendas dos correios sem um ambiente limpo e SEM PUBLICIDADES.


# Stack de técnologia front-end:

1. Jetpack Compose
2. Kotlin
3. MVVM
4. Clean - SOLID
5. Rest Retrofit
4. Kotlin Flows
6. Coroutines 
7. Hilt for dependencie injection
